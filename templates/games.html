Game Instructions :

The objective of the game is to add a series of numbers that are flashed on the screen with a time interval.

Parameters of difficulty :

  1. The digits in each number flashed,
  2. the time interval between two consecutive flashes
  3. the total number of flashed numbers change with difficulty.

There are two modes :

  • Challenge Mode - where we have multiple levels which I created. (BEGINNERS!)
  • Custom mode - where you can set the difficulty parameters (After you get a hang of this!)
